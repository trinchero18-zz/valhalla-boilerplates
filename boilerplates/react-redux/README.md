## Tech stack

React - Redux - Sagas - Reselect - React-router - Styled-components - Axios - Jest - Enzyme

## How to start?

```bash
    viki init <folder_name>
    cd <folder_name>
    yarn install
```

## How to run?

```bash
    yarn install
    yarn start
```

## How to run test suite?

```bash
    yarn test
```
